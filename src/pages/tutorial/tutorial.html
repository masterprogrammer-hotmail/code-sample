Welcome to Injury Finance

The Injury Finance app is a production quality release of the Production Blocks Framework underpinned by the dynamic duo - Ionic and Angular. Production Blocks is backed by the leaders in the mobile cloud space. It is a showcase of talent across the entire Application Lifecycle Management not just a specific stack under development in the Software Development Life Cycle.

What is Ionic?

Ionic Framework is an open source SDK that enables developers to build high quality mobile apps with web technologies like Angular, RxJS, HTML5, CSS3, and TypeScript.

What is Ionic Pro?

Ionic Pro is a powerful set of services and features built on top of Ionic Framework that brings a totally new level of app development agility to mobile dev teams.

This app has been developed using the latest versions of Ionic and Angular and takes advantage of the time saving features of Ionic Pro.
